Tweet.ai

Python Flask Heroku CSS HTML

This web application analyses the last 1000 tweets of the searched topic with a machine learning model in few seconds. Tweets can be either negative, neutral or positive(Sentimental analysis). Twitter has been a social platform for conversation. I thought it is interesting to monitor public opinion on different topics on Twitter. Furthermore, this was a superb opportunity to learn more about web development, data science and visualization.
